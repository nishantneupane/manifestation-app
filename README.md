# Manifestation App

A simple web application designed to help users visualize and track their goals through manifestation techniques.

## 🚀 Features
- Set and track personal goals
- Daily affirmations
- Visualization tools
- Progress tracking dashboard

## 🛠️ Tech Stack
- Frontend: Next.js / React
- Backend: Node.js
- Database: (Add yours here — e.g., MongoDB / Supabase)

## 📦 Installation

```bash
git clone https://github.com/nishantneupane/manifestation-app.git
cd manifestation-app
npm install
npm run dev
```

## 📌 Usage
Open your browser and go to:
```
http://localhost:3000
```
